# Общие данные БД (Остров + бот)

Цель: один источник правды для пользователя и между вебом и ботом.

## Сейчас

- Пользователи и мечты — таблицы Острова (см. `tables.md` в репо Острова).
- Бот дублирует часть «памяти» в файлах `user_profiles/` на сервере с Bloom — **временно**; долгосрочно перенос в БД Острова (отдельное проектирование).

## Планируется

- `users.telegram_id` — связка Telegram ↔ аккаунт.
- `daily_report_log` (или аналог) — антидубли вечернего отчёта.
- Таблицы/поля для biography / хвост чата / bloom_traits — **TBD**, согласовать здесь после миграции.
