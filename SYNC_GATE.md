# SYNC_GATE

Обязательный шлюз синхронизации для Продагента и Bridge перед любыми действиями.

## Зачем

- Убрать рассинхрон ролей и путей.
- Гарантировать, что все агенты работают по одному SSOT (`island-bridge-contract`).

## Обязательная процедура (до любых изменений)

1. Обновить SSOT:
   - `cd /home/makc/Apps/island-bridge-contract`
   - `git pull --ff-only origin main`
2. Прислать контроль:
   - `git rev-parse --short HEAD`
   - дословную строку из `THE_LOFT.md` про `FORGE SANDBOX`
3. Ответить в формате:
   - `SYNC_OK <hash>`
   - `FORGE_SANDBOX: <цитата из THE_LOFT.md>`

Без этих двух строк любой ответ считается недействительным.
